# Session Logging Protocol

## Trigger

End of every working block

---

## Minimum Required Entry

- Date
- Project
- Objective
- Assets Created
- Decisions Made
- Next Actions

---

## Compression Rule

- Bullet points only
- No fluff
- No explanations

---

## Example

Objective:
- Fix onboarding flow

Assets Created:
- Service selector fix
- Validation logic patch

Decisions:
- Prioritize onboarding over UI polish

Next Actions:
- QA onboarding
- Deploy fix

---

## Target Time

< 60 seconds
