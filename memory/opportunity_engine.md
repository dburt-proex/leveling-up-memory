# Opportunity Engine (Operational Use)

## When to Run

At start of every session

---

## Inputs

- Active projects
- Existing assets
- Bottlenecks
- Monetization gaps

---

## Output

Generate exactly 3 options:

OPTION 1:
- Title
- Outcome
- Why it matters
- Effort
- Leverage Type

OPTION 2:
(same)

OPTION 3:
(same)

---

## Selection Criteria

1. Fastest path to revenue
2. Highest leverage per effort
3. Asset compounding potential

---

## Rule

After selection:
→ Execute immediately
